# Marketing Data Query App

A modern Next.js application for querying and analyzing marketing data from multiple sources.

## Description

This project is a comprehensive marketing data query application built with Next.js, TypeScript, and Tailwind CSS. It provides a powerful dashboard for analyzing marketing data from Google Analytics, Facebook Ads, Twitter, LinkedIn, and other platforms.

## Getting Started

### Prerequisites

- Node.js 18+ 
- npm or yarn
- Git

### Installation

1. Clone the repository:
```bash
git clone <repository-url>
cd maketing-data-query
```

2. Install dependencies:
```bash
npm install
# or
yarn install
```

3. Set up environment variables:
```bash
cp .env.example .env.local
# Edit .env.local with your API keys and configuration
```

4. Run the development server:
```bash
npm run dev
# or
yarn dev
```

5. Open [http://localhost:3000](http://localhost:3000) in your browser.
